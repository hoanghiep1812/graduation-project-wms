# Warehouse Management System (EasyWMS)

A smart warehouse management system developed for graduation project purposes.  
The system supports warehouse operations, inventory tracking, analytics, mobile integration, and AI chatbot assistance.

---

## Overview

This project was developed to support warehouse management activities including:

- Product management
- Warehouse import/export operations
- Inventory tracking
- ABC inventory analysis
- Statistical dashboard
- RESTful API integration
- Mobile application support
- AI Chatbot with RAG support

---
